For the first part I used CSS media queries and viewport to implement a fluid response for three different breakpoints. I found the use of media queries very useful. I liked the fact that I could change with the classes based on the viewport Making the page more responsive was fairly straight forward, thoough having to build each column and row separatly was a little bit of a hassle however it helped me understand what each part was doing and made me feel more in control of what I was changing.
